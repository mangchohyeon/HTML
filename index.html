HTML

HTML : 하이퍼 텍스트 마크업 언어
Hyper Text Markup Language의 약자
HTML은 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어이다.




CSS

CSS : 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어
Style sheet language의 약자
레이아웃과 스타일을 정의할 때 사용되는 언어이다.




Javascript

JavaScript : 자바스크립트는 객체지향 프로그래밍 언어이다.
이 언어는 웹 브라우저 내에서 주로 사용된다.
또한 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.

연습용으로 쓸 HTML파일 박건이 만듬.
