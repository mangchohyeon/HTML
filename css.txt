인라인 스타일 : style 속성
<nav style : 'background-color : aliceblue'
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

내부 스타일 : style 태그 이용(html 코드안에서 style을 작성)
<style>
        nav {
            background-color: aliceblue;
        }
    </style>
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

외부 스타일 : css 파일을 통해 style을 정의
nav {
            background-color: aliceblue;
        }
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

그룹화 선택자(Grouping Selector) : 같은 스타일을 여러 요소에 적용시키는 것
nav, li {
    background-color : aliceblue;
}
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

복합 선택자(Combinator Selector)

자식 선택자(Child Combinator) : 선택자 A > 선택자 B
선택자 A안에 존재하는 선택자 B에 스타일을 적용

후손 선택자(Descendant Combinator) : 선택자 A 선택자 B


일반 형제 선택자(General Sibiling Selector) : 선택자 A ~ 선택자 B

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

가상 클래스 선택자

HTMl 요소의 상태나 위치에 따라 특정 스타일을 적용
가상 클래스는 HTML문서의 DOM(Document Object Model) 구조에서
실제로 존재하지 않는 상태를 나타내며, 요소가 특정 상태에 있을 때 스타일을 적용
요소의 특정 상태에 따라 스타일을 정의할 때 사용

link : 셀렉터가 방문하지 않은 링크일 때
visited : 셀렉터가 방문한 링크일 때
hover : 셀렉터에 마우스가 올라와 있을 때
active : 셀렉터가 클릭된 상태일 때
focus : 셀렉터에 포커스가 들어와 있을 때
